# Broadcast Channel — Agent Ecosystem

## What This Is
This file is a **one-to-many broadcast** for the Agent Coordination Protocol (ACP).

When Philip wants to ask ALL agents the same question simultaneously, he writes it here.
Every agent sees it — Claw, Hermes, Copilot, any future agents.

## How It Works

1. **Philip writes a question here**
2. **Fan-out script distributes** the question to every agent's personal inbox
3. **Each agent processes it independently** during their next session
4. **Agents append their response** to their outbox.md
5. **Gateway routes responses** back to Philip (or to each other)

## Rules
- One question per `###` block
- Mark status: `[PENDING]` → `[ACK: hermes]` → `[ANSWERED by: claw, hermes]` → `[CLOSED]`
- Agents do NOT edit this file — they edit their own inbox/outbox
- Only Philip and the fan-out script touch this file
