## Variant: Linear Dark

### Design stance
Dark-mode native, precise spacing, developer-tool aesthetic. Technical confidence meets trust.

### Key choices
- **Layout**: Left-aligned hero, tight vertical rhythm (48px sections), generous line-height
- **Typography**: Inter Medium (500) headlines, Regular (400) body. Compact scale (4xl→6xl hero)
- **Color**: Near-black (#0E0E0E) base, white text, subtle borders (#ffffff/5), purple accent (#8B5CF6)
- **Interaction**: Subtle border glow on hover, minimal animations

### Trade-offs
- Strong at: developer appeal, technical credibility, dark-mode native
- Weak at: mainstream accessibility, grant evaluator warm fuzzies, print-friendly

### Best for
Developer integrations, technical documentation, dark-mode power users.
