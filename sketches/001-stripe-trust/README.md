## Variant: Stripe Trust

### Design stance
Generous spacing, weight-300 typography, editorial authority. Trusted institution meets blockchain.

### Key choices
- **Layout**: Centered hero, generous whitespace (56px+ vertical rhythm)
- **Typography**: Inter Light (300) headlines, Regular (400) body. Large type scale (5xl→7xl hero)
- **Color**: White base, deep navy (#0A2540) text, purple gradient accent (#635BFF→#A78BFA)
- **Interaction**: Hover lifts on cards, smooth shadow transitions

### Trade-offs
- Strong at: conveying trust, institutional credibility, grant evaluator appeal
- Weak at: developer excitement, technical density, dark-mode preference

### Best for
Grant applications, investor due diligence, partnership proposals.
