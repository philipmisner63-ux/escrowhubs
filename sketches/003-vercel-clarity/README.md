## Variant: Vercel Clarity

### Design stance
Geometric precision, block-based layout, chain-forward developer aesthetic. Technical infrastructure company.

### Key choices
- **Layout**: Left-aligned, rigid grid, generous gutters. Hero is a statement block, not a swoosh
- **Typography**: Geist Sans (geometric, compressed), Geist Mono for labels. Medium weight throughout
- **Color**: White (#fff) base, pure black (#000) text, minimal accent (green for live status only)
- **Interaction**: Subtle border-color transitions, pill-shaped chain badges, geometric buttons (6px radius)

### Trade-offs
- Strong at: developer credibility, technical infrastructure positioning, fast scanning
- Weak at: emotional warmth, mainstream accessibility, brand personality

### Best for
Technical audiences, developer tools, infrastructure companies seeking clarity over warmth.
